# Code of Conduct

AI Flow Builder aims to be a practical, respectful open source project.

## Expected Behavior

- Be direct, constructive, and specific.
- Focus technical disagreement on code, behavior, architecture, and user impact.
- Respect different experience levels and communication styles.
- Keep discussions scoped to the project and the issue at hand.
- Assume that contributors are working in good faith, while still asking for evidence and tests when needed.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Personal attacks or sustained disruption of technical discussions.
- Publishing private information without permission.
- Sexualized content or attention in project spaces.
- Deliberately introducing malicious code, credential leaks, or unsafe deployment guidance.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code. Maintainers may also temporarily or permanently restrict participation when necessary to protect the project.

## Reporting

Report conduct concerns through the security contact described in [SECURITY.md](SECURITY.md), or by opening a private maintainer communication channel if one is available for the repository.
